## Installation

- Android studio 3.4 or higher
- Gradle 5.6 or higher
- Java SDK 1.8
- Android API Level 21 (Lollipop) or higher


### Clone

- Clone this repo to your local machine using `https://github.com/startng/firestorm-personal-fin-tracker.git`
